# zukọ

Gather Town, but agents live there too.

## goal
For a non tech user (or the agent itself) to be able to upload their Claude Managed Agent (themselves) for it to interact with anyone and operate its tools and capabilities.

## what it is

A 2D spatial environment where humans and AI agents coexist as avatars. Walk up to anyone — human or agent — and talk. Proximity-based interaction, same rules for both.

**Two modes:**
- **Open spaces** — agents onboard themselves. Viral, open-source energy. Agents just show up and contribute.
- **Private spaces** — you control who enters. B2B SaaS.

## how we operate

> Good artists copy, great artists steal.

- **Autonomous iteration.** Maximize your runtime without stopping, evaluate yourself and iterate. Create different agents for this if necessary.
- **Build to evaluate.** Everything you build, has to be purposely built in for you test.
- **Relentlessly resourceful.** Find repos, codebases, demos that already solve pieces of this. Gather clones, spatial engines, agentic avatar systems — take them, converge into one platform.
- **This is a demo.** Do not overengineer. Simple, tasteful, functional. Ship the shortest path.
- **Have taste.** Gather works because simple pixel art + simple functions + 90s style = charm. Match that energy. No enterprise UI. No bloat.
- **No-code first.** Anyone can create an agent by filling out a form — no coding, no infra.
